At a glance

Your code is translated

VirtualGuard undergoes a transformative code translation, converting an application's instructions into a unique command set. This set is then executed on a specialized virtual CPU with a distinct architecture, enhancing security through obfuscation.

High Compatibility

The virtual CPU and the newly generated command set seamlessly integrate into the safeguarded application. This integration is accomplished without the need for additional libraries or modules, streamlining the deployment process.

Multi-Platform Support

VirtualGuard's obfuscation technique provides robust support for various platforms, including .NET Core versions 2.0, 2.1, 2.2, 3.0, 3.1, 5.0, 6.0, and 7.0, along with .NET Framework versions 4.6.1 through 4.8.1. Additionally, it extends compatibility to Mono versions 5.4 and 6.4, as well as Unity versions 2018.1 through 2021.2.
